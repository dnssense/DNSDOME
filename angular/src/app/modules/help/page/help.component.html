Frequently Asked Questions
How Can I Define New Agent? keyboard_arrow_down
In order to define an agent: go to Network-> Public IP menu. Then click "Create New Agent" button on top right corner. Define your own agent information. Then select a predefined security profile or create a new profile. Finally save it.
How Can I Define Security Profiles? keyboard_arrow_down
In order to define security profile: go to Settings-> Security Profiles menu. Then define your own security profile with a name.
How Can I activate Two Factor Authentication (2FA)? keyboard_arrow_down
Go on top left corner and click on your username. Then go to Account Settings menu. You can see "Enable two factor Authentication" button in User Information panel.
