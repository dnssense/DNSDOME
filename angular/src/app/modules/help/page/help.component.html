Frequently Asked Questions
How DNS Forwarding Works? keyboard_arrow_down
The DNS is the address book of Internet. All protocols require DNS to connect to Internet. Applications can not communicate if they can not resolve the IP address of the domain they want to access. Because of this providing security over DNS is very effective. DNSCyte Secure DNS service handles DNS requests from its users and redirects malicious requests that are not allowed to access to the DNSCyte sinkhole IP address. This way, harmful traffic is stopped before it even begins.
How Can I implement DNS Forwarding? keyboard_arrow_down
  1. Open an account from portal.dnscyte.com
  2. Click the "Register" button at https://portal.dnscyte.com and fill out the form. All you need is to register an email address.
  3. Identify your external IP Address and security policy from the portal.
  4. The DNSCyte user portal is designed to be extremely easy to configure. Use Deployment -> Public IP menu to define the external address.
  5. Select a profile from the profile list that you think is appropriate, or click "New Profile" to create a new profile
  6. Save your settings by clicking the Save Button.
  7. Enable devices on your network to use DNSCyte DNS Servers. Steps to enable this is based on how DNS is configured in your organisation.
    • - DNSCyte DNS Servers are 199.244.90.90 and 199.244.90.91
    • - If you use a local DNS server, you must define DNSCyte DNS IP Addresses as forwarder DNS on your local DNS Server.
    • - If you do not use Local DNS, you must deploy DNSCyte DNS IP addresses as the DNS Server IP Address over DHCP.
  1. Please refer “DNSCyte Secure DNS Setup Guide(PDF)” for more details. You can download it from here
  2. For roaming clients you can download the Windows Client from Deployment -> Roaming Client menu inside the GUI.
How Can I Define New Agent? keyboard_arrow_down
In order to define an agent: go to Network-> Public IP menu. Then click "Create New Agent" button on top right corner. Define your own agent information. Then select a predefined security profile or create a new profile. Finally save it.
How Can I Define Security Profiles? keyboard_arrow_down
In order to define security profile: go to Settings-> Security Profiles menu. Then define your own security profile with a name.
How Can I activate Two Factor Authentication (2FA)? keyboard_arrow_down
Go on top left corner and click on your username. Then go to Account Settings menu. You can see "Enable two factor Authentication" button in User Information panel.
